Cheat Sheet - Math

Back to Cheat Sheet Directory

Summary

Computers are VERY good at math. That's why we need to learn about using them for this purpose. This cheat sheet will show you the basic tools Java provides you with in order to do math effectively.

Basic Operations

Most operations with numbers are just how we write it in real life. Here is how to add, subtract, multiply, divide, or find the remainder for two INTEGERS.

int a = 5;
int b = 3;

System.out.println(a + b); // 8
System.out.println(a - b); // 2
System.out.println(a * b); // 15
System.out.println(a / b); // 1
System.out.println(a % b); // 2 (the remainder of a / b is 2)

You can always just set the result of some math to a variable, but make sure it is the right type. This will be covered futher down.

int result = a + b;

Basic Operations - Decimals

When you are working with decimals, you need to be careful. Java will automatically round down to the nearest integer if you are not careful. Here is how to add, subtract, multiply, divide, and find the remainder of two DECIMALS.

double c = 5.1;
double d = 3.2;

System.out.println(c + d); // 8.3
System.out.println(c - d); // 1.9
System.out.println(c * d); // 16.32
System.out.println(c / d); // 1.59375
System.out.println(c % d); // 1.9 (the remainder of c / d is 1.9)

Just like with integers, you can set the result of some math to a variable, but make sure it is the right type. What happens, though, if you set the result to an integer instead of a double?

double result = c + d; // 8.3
int result = c + d; // 8

As you can see, you lose information if you store a decimal value as a double. Make sure to be careful when storing your variables!

Extra Math Features

Mostly, for the rest of this cheat sheet, we will be working with doubles as that's just how these functions work. The built-in Math class in Java gives us tons of resources to do complex math and math-adjacent things.

To get the absolute value of a number:

double n = -5.1;
System.out.println(Math.abs(n)); // 5.1

To get the mathematical constant PI:

System.out.println(Math.PI); // 3.141592653589793

To get the sin, cos, or tan of a ratio:

double angle = Math.PI / 4;
System.out.println(Math.sin(angle)); // 0.7071067811865475
System.out.println(Math.cos(angle)); // 0.7071067811865476
System.out.println(Math.tan(angle)); // 0.9999999999999999

To round numbers up, down, or normally:

double number = 5.6;
System.out.println(Math.ceil(number)); // 6.0
System.out.println(Math.floor(number)); // 5.0
System.out.println(Math.round(number)); // 6

To get the minimum or maximum of two numbers:

double x = 5.6;
double y = 3.2;
System.out.println(Math.min(x, y)); // 3.2
System.out.println(Math.max(x, y)); // 5.6

To raise a number to a power:

double base = 2;
double exponent = 3;
System.out.println(Math.pow(base, exponent)); // 8.0

To get the square root of a number:

double number = 9;
System.out.println(Math.sqrt(number)); // 3.0
//or
System.out.println(Math.pow(number, 0.5)); // 3.0

And lastly, to get a random number between 0 and 1:

System.out.println(Math.random()); // 0.123456789

Don't worry, random values will become much more useful later on.

Changing numbers in one line

When you want to change a number in one line, you can use the following shorthand. This is especially useful when you want to increment or decrement a number by one.

int number = 5;
number++; // number is now 6
number--; // number is now 5

But what if you want to increment or decrement by more than one? You can use the following shorthand.

int number = 5;
number += 3; // number is now 8
number -= 2; // number is now 6

This also goes for all the other basic operators. Instead of adding or subtracting, you can multiple, divide, or modulus!

int number = 5;
number *= 3; // number is now 15
number /= 5; // number is now 3
number *= 10; // number is now 30
number %= 7; // number is now 2 (30 / 7 leaves a remainder of 2)

Checking divisibility

Using modulus, you can find out if something is divisible by another thing. If you modulus one number with another, if the answer is 0, then it must be divisible by that number, as there is no remainder! For example:

int number = 10;
int divisor = 5;
System.out.println(number % divisor); // 0, so 10 is divisible by 5

But if you modulus a number that is not divisible by another, you will get a remainder. For example:

int number = 10;
int divisor = 3;
System.out.println(number % divisor); // 1, so 10 is not divisible by 3

Using this, you can check if a number is even or odd. If a number modulus 2 is 0, then it is even. If it is 1, then it is odd.

int number = 10;
int otherNumber = 15;
System.out.println(number % 2); // 0, so 10 is even
System.out.println(otherNumber % 2); // 1, so 15 is odd
