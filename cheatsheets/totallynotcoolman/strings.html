Cheat Sheet - Strings

Back to Cheat Sheet Directory

Summary

Strings are the way Java stores text- and actually, how most languages store text. This cheat sheet will inform you how to create strings, use escape sequences, and concatenate them as well.

Creating Strings

To create a normal string, you simply put it between quotes. You can set it to a variable, or directly use it in a print statement.

System.out.println("Hello, World!");

// or
String message = "Hello, World!";
System.out.println(message);

Escape Sequences

In Java, the way we type certain special characters is using escape sequences. This goes for characters that aren't exactly on your keyboard- like the newline or the tab. It also lets you type characters that mean something important- like quotes or backslashes. Here are some examples.

System.out.println("Newline: \n");
System.out.println("Tab: \t");
System.out.println("Quotes: \"");
System.out.println("Backslash: \\");

Using these escape sequences, we can type in things that aren't exactly normal. We mainly use newlines with this method, because it lets us type multiple lines of text without using new print statements. For example:

System.out.println("What is 1+1?\n1+1=2");
That will put the line "1+1=2" onto the next line in the console, or whatever other method you use to display it.

Concatenation

If you want to add strings together, you concatenate them. To concatenate two strings, do the following:

String first = "Hello, ";
String second = "World!";
String message = first + second; // Hello, World!

You can also concatenate strings and numbers.

int number = 5;
String message = "The number is " + number; // The number is 5

But what doesn't work is trying to do math with concatenation. Just like with PEMDAS, you would need parenthesis, or the numbers would be put into the string, like so.

// Does NOT work
int number = 40;
String message = "The number is " + number + 2; // The number is 402

// Works
int number = 4;
String message = "The number is " + (number + 2); // The number is 42

This happens because Java concatenates things left to right, so for the non working example, instead of adding number and 2 then adding that to the string, it adds number to the string and then 2 to the new string.
